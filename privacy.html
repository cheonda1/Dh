개인정보 처리방침

WBMoveSequencer

← 메인으로 돌아가기
WBMoveSequencer(이하 "앱")은 사용자의 개인정보를 일절 수집하지 않습니다.

본 앱을 사용하는 동안 어떠한 정보도 서버에 저장하거나 외부로 전송하지 않으며, 사용자의 프라이버시 보호를 최우선으로 생각합니다.

1. 개인정보 수집 항목

본 앱은 어떠한 개인정보도 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보 수집 없음
  • 위치 정보 수집 없음
  • 기기 정보 수집 없음
  • 사용 패턴 추적 없음

2. 데이터 저장 및 사용

앱에서 생성하는 모든 데이터(포즈, 모션, 이미지 등)는 사용자의 기기에만 저장됩니다.

  • 모든 데이터는 사용자의 iPhone/iPad 로컬 저장소에 보관
  • 외부 서버로 데이터 전송 없음
  • 클라우드 동기화 없음
  • 제3자와의 데이터 공유 없음

3. 사진 라이브러리 접근

앱은 사용자가 이미지를 저장할 때 사진 라이브러리 쓰기 권한을 요청합니다.

  • 이미지 저장 기능에만 사용
  • 기존 사진 읽기 권한 요청하지 않음
  • 사용자가 명시적으로 저장 버튼을 눌렀을 때만 작동
  • 저장된 이미지는 사용자의 완전한 통제 하에 있음

4. 네트워크 통신

본 앱은 완전한 오프라인 앱입니다.

  • 인터넷 연결 불필요
  • 서버와의 통신 없음
  • 광고 네트워크 없음
  • 분석 도구(Analytics) 없음

5. 제3자 서비스

본 앱은 어떠한 제3자 서비스도 사용하지 않습니다.

  • 광고 플랫폼 미사용
  • 분석 서비스 미사용
  • 소셜 미디어 SDK 미사용
  • 결제 시스템 미사용 (앱은 무료입니다)

6. 아동의 개인정보 보호

본 앱은 만 13세 이상의 사용자를 대상으로 하며, 만 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 어떠한 개인정보도 수집하지 않으므로 연령과 무관하게 안전하게 사용할 수 있습니다.

7. 데이터 보안

모든 데이터는 사용자의 기기에만 저장되므로, iOS의 샌드박스 보안 시스템에 의해 보호됩니다.

  • 다른 앱이 본 앱의 데이터에 접근할 수 없음
  • iOS 백업 기능을 통해 데이터 백업 가능
  • 앱 삭제 시 모든 데이터가 함께 삭제됨

8. 개인정보 처리방침 변경

본 개인정보 처리방침은 법령, 정책 또는 보안 기술의 변경에 따라 내용이 수정될 수 있습니다. 개인정보 처리방침이 변경되는 경우, 앱 업데이트를 통해 공지하겠습니다.

9. 문의

개인정보 처리방침에 대한 문의사항이 있으시면 개발자에게 연락해 주시기 바랍니다.

이메일 또는 App Store 개발자 연락처를 통해 문의 가능합니다.
